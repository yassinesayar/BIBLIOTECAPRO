Este proyecto consiste en una aplicación desarrollada en Java que permite gestionar libros y usuarios dentro de un sistema sencillo.
Se aplican conceptos de Programación Orientada a Objetos (POO) como:
Clases y objetos
Encapsulamiento
Enumeraciones (enum)
Separación de responsabilidades
El sistema permite crear libros, asignarles un género y estado, así como crear usuarios con su respectivo estado.
